# Standard Skills

我们是为 Agent Skill 写标准的工作室。

- **skill-intelligence-framework/** — 分析和设计 Skill 的元 Skill
- **cascade-review/** — 强模型用的并行代码审查
- **local-code-review/** — 弱模型用的模式匹配审查
- **local-code-gen/** — 弱模型用的模板代码生成

首次使用建议先读 `skill-intelligence-framework/`。
