# skill-types.md — DEPRECATED

This file has been replaced by the canonical 8-archetype system in SKILL.md and references/archetype-router.md.

Do not use this file for new skill evaluations.

## Canonical Archetype System (8 + Verification)

SKILL.md defines 8 archetypes: Diagnostic, Delivery, Stance, Director, Workflow, Pipeline, Knowledge, Research.
Verification is a separate cross-cutting archetype with its own guide.

## Old → New Mapping

| Old (this file) | New (canonical) | Notes |
|---|---|---|
| Diagnostic | Diagnostic | unchanged |
| Delivery | Delivery | unchanged |
| Workflow | Workflow | unchanged |
| Generator | Delivery | Generator is a Delivery subtype (artifact-producing). No longer a separate archetype. |
| Director | Director | unchanged |
| Integrator | Pipeline | Same concept, renamed to Pipeline for clarity. |

## Missing from Old File

The following canonical archetypes have no equivalent in the old system:
- Stance (was not distinguished from other types)
- Knowledge (was not recognized as a separate archetype)
- Research (was not recognized as a separate archetype)
- Verification (was not recognized as a separate archetype)

## Why Deprecated

The old 6-type system did not distinguish Stance from Procedure, conflated Pipeline with Workflow, and was missing Knowledge and Research archetypes that are necessary for the current framework.
