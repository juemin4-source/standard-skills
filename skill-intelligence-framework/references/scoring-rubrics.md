# Scoring Rubrics

## Discovery Score (0-50)
- thought density ×3
- suspicion strength ×2
- probe quality ×2
- open-ended discovery ×2
- root-cause seeking ×2
- failure model depth ×2
- search breadth ×1
- penalties: probe closure -2, over-formatting -1

## Delivery Score (0-50)
- output structure ×3
- priority ordering ×2
- clarity ×2
- actionability ×2
- tone control ×1
- specificity ×1
- reusability ×1
- penalties: vague recommendations -2, no prioritization -1

## Integration Score (0-50)
- trigger clarity ×2
- scope boundaries ×2
- context efficiency ×2
- tool/file awareness ×1
- safety ×1
- composability ×1
- low misfire risk ×1
- penalties: overbroad description -2, hidden assumptions -1

## Generation Score (0-50)
- constraint satisfaction ×3
- output completeness ×2
- iteration support ×2
- quality criteria ×2
- adaptation ×1
- format correctness ×1
- verification ×1

## Director Score (0-50)
- core thesis ×3
- taste/value position ×2
- priority logic ×2
- tradeoff handling ×2
- false-goal rejection ×2
- root-cause identification ×2
- substance-vs-polish ×1

## Wisdom Density (independent 1-10)
Single measure: how much expert judgment is compressed into how little text.
