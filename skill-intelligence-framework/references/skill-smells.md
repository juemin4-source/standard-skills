# Skill Smells

## Procedure Without Judgment
Steps exist but no decision framework. Agent follows orders without understanding why.

## Checklist Without Priority
Items are listed but nothing is ranked. Everything seems equally important.

## Output Format Mistaken For Expertise
Pretty templates that look professional but contain no real analysis.

## Overbroad Scope
Tries to do everything, does nothing well. No boundary awareness.

## Vague Trigger
Description is so generic the skill fires on unrelated tasks.

## Generic Excellence Language
Uses words like "high quality", "best practice", "professional" without defining them.

## Style Words Replacing Criteria
"Elegant", "beautiful", "clean", "robust" used instead of measurable standards.

## Too Much Philosophy, Not Enough Probes
Talks about what matters but never tells the agent what to look for.

## Probe Closure
Probes are written as exhaustive boundaries. Agent stops searching beyond them.

## No Failure Model
Does not know what bad outcomes look like. No anti-patterns, no counterexamples.

## No Verification Step
No way to tell if the skill's output is correct or useful.

## No Boundary Awareness
Does not state what it is not for. Gets applied to wrong tasks.
